ttrss

ttrss is a Tiny Tiny RSS Reader App for the Nokia N9 smart phone, written using Qt/QML. It uses the Tiny Tiny RSS API.

If you want to contribute, feel free to use the Issue Tracker or send me a Pull Request.

If you want to discuss, there is the official Maemo.org Thread.
