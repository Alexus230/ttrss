ttrss
==========

ttrss is a [Tiny Tiny RSS](http://tt-rss.org) Reader App for the
Nokia N9 smart phone, written using Qt/QML.
ttrss is using the [Tiny Tiny RSS API](http://tt-rss.org/redmine/projects/tt-rss/wiki/JsonApiReference).
